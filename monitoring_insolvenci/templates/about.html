O aplikaci

Tato webová aplikace byla vytvořena v rámci bakalářské práce. Jejím cílem bylo představit vývoj webových aplikací v jazyce Python 3.

Apliakce umožňuje uživatelům sledovat obchodní partnery a jejich insolvenční řízení. Data partnerů jsou odebírána s veřejného zdroje ARES. Data insolvenčních řízení jsou odebírána z veřejného zdroje Insolvenční rejstřík.

Použité technologie

Aplikace využívá v maximálních možnostech jazyk Python 3 s frameworkem Flask a šablonovacím jazykem Jinja2 nad HTML.

Pro moderní vzhled jsou použity CSS a JS knihovny Bootstrap a Datatables.
